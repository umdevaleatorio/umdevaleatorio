## 📱 Destaque de Engenharia: AgroPet Lambari

O **AgroPet Lambari** é o meu principal projeto prático acadêmico, onde apliquei rigorosamente os padrões de engenharia modernos. Consiste em um ecossistema mobile com **2 APKs isolados** (Cliente e Admin) operando em tempo real.

### 🌟 Pilares de Engenharia do Projeto:
1. **Offline-First (SQLite + Supabase):** Catálogo persistido localmente para carregamento instantâneo e operação resiliente de carrinho 100% offline. Sincronização segura via Supabase Database.
2. **Validação Atômica de Checkout (Supabase RPC):** Proteção transacional ACID usando `SELECT ... FOR UPDATE` no PostgreSQL para evitar concorrência desleal em itens de estoque limitado, integrado a triggers automáticos de ativação/desativação de catálogo.
3. **Telemetria GPS Realtime:** Serviço reativo de posicionamento do entregador (`expo-location` em background no app Admin) persistindo dados no Supabase Realtime, consumidos via WebSockets pelo cliente com **interpolação e suavização contínua** no mapa.
